## 丨简介
纳西妲计算器是一个用于角色伤害计算以及圣遗物评分的系统
## 丨安装方法
将本项目直接clone或者下载压缩包解压到`LittlePaimon\plugins\`内即可
## 丨指令
* upd<角色名> : 在小派蒙获取信息后，更新虚空内的信息
* nhd重置：清楚虚空中的玩家信息
* nhd<角色名>：查看面板详情
* set<角色名>：设置某位角色的信息
